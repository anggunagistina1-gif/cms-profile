Tentang Website 🌸

Saya Anggun Agistina 👋 disini saya membuat CMS sederhana untuk memenuhi tugas responsi mata kuliah web programming.


Website ini menggunakan HTML, CSS, dan JavaScript dengan penyimpanan data menggunakan localStorage.


Website ini mengelola artikel dengan fitur CRUD (Create, Read, Update, Delete).


Pengguna dapat menambahkan artikel melalui dashboard, kemudian artikel akan disimpan dan ditampilkan kembali pada halaman artikel.
