# CMS Profile - Anggun Agistina

Nama : Anggun Agistina
NIM : 24EO10011
Prodi : Informatika

## Deskripsi

Aplikasi CMS Profile berbasis HTML, CSS, JavaScript, PHP, dan MySQL.

## Fitur

* Login menggunakan Local Storage
* Manajemen Artikel CMS
* Integrasi Artikel ke Halaman Profil
* Map Location menggunakan Geolocation API dan Leaflet
* Service Worker API
* Push Notification

## Struktur Folder

Frontend

* cms.html
* login.html
* profil.html
* sw.js
* asset/

Backend

* tambahArtikel.php
* hapusArtikel.php
* test.php

Database

* database.sql

## Cara Menjalankan

1. Jalankan Apache dan MySQL pada XAMPP.
2. Import database.sql ke phpMyAdmin.
3. Simpan folder proyek di htdocs.
4. Buka aplikasi melalui localhost.
